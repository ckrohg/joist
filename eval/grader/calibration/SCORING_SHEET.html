Joist Clone — BLIND Calibration Scoring Sheet

Score each source→clone pair the way a real visitor would perceive it. You are the human ground truth. The grader's current scores are deliberately hidden so they don't anchor you.

Scale for every axis: 0 = totally wrong / broken  ·  100 = indistinguishable from the source. Whole numbers, 0–100.
Pixel-fidelity
Does the static look match? Layout, color, type, imagery, completeness — what a screenshot would show.
Responsive
Does it hold up at the shown width (esp. the mobile pairs)? Reflow, no blank/empty bands, no overflow.
Motion / interaction
Hover states, scroll-reveal, marquees, parallax. If the source clearly animates and the clone is static, score low. If neither side moves, score what you'd expect a visitor to feel — leave blank only if truly unknowable from a still.
Editability
Best-guess: if you opened this in a page builder, are these real editable text/blocks — or a frozen image / baked-in screenshot? Judge from artifacts (crisp text vs blurry raster, clipping, dead regions).
Overall
Your single gut verdict: would you ship this as a clone of the source? Not an average — your holistic call.
