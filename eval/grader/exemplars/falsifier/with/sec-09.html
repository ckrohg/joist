CSS variables

Customizing your theme is as simple as creating a few CSS variables.

 1  @theme {
 2    --font-sans: "Inter", sans-serif;
 3    --font-mono: "IBM Plex Mono", monospace;
 4    --text-tiny: 0.625rem;
 5    --color-mint-100: oklch(0.97 0.15 145);
 6    --color-mint-200: oklch(0.92 0.18 145);
 7    --color-mint-300: oklch(0.85 0.22 145);
 8    --color-mint-400: oklch(0.78 0.25 145);
 9  }

Cascade layers

Tailwind uses CSS layers so you don't have to worry about specificity issues.

 1  @layer theme, base, components, utilities;
 2  @layer theme {
 3    /* your theme variables */
 4  }
 5  @layer base {
 6    /* preflight styles */
 7  }
 8  @layer components {
 9    /* your component classes */
10  }
