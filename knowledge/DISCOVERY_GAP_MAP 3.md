## Honest summary, per dimension

The clone is **geometrically excellent and structurally thin**. Raw per-element placement is effectively solved (pos/cov recovers 0.967–0.984 on all 7 sites; hRatio 1.000); every low `position` number is an **areaCoverage (under-capture) artifact**, not drift — so re-attacking placement directly is wasted motion. **Color (0.42 corpus)** splits into two populations: a genuine large-area **dark-band-captured-as-`bg:None`** paint defect concentrated on motion-heavy sites the router sends to **build-hybrid** (framer/resend, meanDE 14–37), and an areaCoverage-collapse artifact on thin-capture sites (supabase/reactdev/vercel) that is NOT a paint bug. **Typography (0.55)** is dragged by one deterministic regex bug — `gFont('sans-serif') → 'Georgia'` (verified in isolation) — that renders ~118 framer nav/CTA leaves (and react.dev/vercel generic-stack text) in a serif on sans-serif sites: the most jarring, highest-attention font error possible, and a one-line fix. **Structural (0.55)** has two whole categories systematically zeroed: **forms never detected end-to-end** (verified: zero `kind:'form'` in capture or build; missed on 5/7 sites — the only miss on reactdev and vercel) and framer CSS-`<video>` lands 0/20 (capture-side, riskier). **Editability (0.62 honest / perElement.text 0.24–0.50 on five sites)** is throttled by a `flat.length >= 40` hard cap in the deep-flatten path (verified at capture-layout.mjs:1014–1017) — vercel's two biggest sections match at *exactly* 39 and 40, a dead-on fingerprint — silently dropping every text leaf past the 40th into `unmatchedSrcArea`, which multiplicatively craters text/color/typo/position. **Imagery (0.52)** loses text-bearing mid-width feature cards (resend 510–602px, framer 360–550px) to the mockup rasterizer because the text-band override demands `mb.w >= 1000` (verified line 714). The remaining #1 wall (deep-text-leaf coverage on linear/vercel/reactdev; JS-rendered code blocks) is correctly **SUPERVISED** (container-inference build-flow.mjs + native Code-Highlight widget, already queued) — not manufactured here.
